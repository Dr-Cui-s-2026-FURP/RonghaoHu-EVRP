# Node 1: 文献综述 —— 电动车辆路径规划中的充电与换电策略差异

## 1. 引言

电动车辆的有限续航和较长的充电时间引发了“里程焦虑”（Range Anxiety），成为制约其大规模应用的核心瓶颈。为应对这一挑战，研究者提出了两种主要的能源补给策略：

- **充电（Charging）**：车辆在充电站停留较长时间，电量按非线性速率增加。
- **换电（Swapping）**：车辆在换电站快速更换整块电池，耗时短且电量瞬时恢复至满值。

说人话即是，充电是不对电池进行更换，而直接充电；换电则倾向于更换通用电瓶。换电耗时一般较短（除非使用电容储电），但是存在大量技术问题和金钱成本代价。

这两种策略在路径规划问题中导致完全不同的数学结构与决策逻辑。本综述旨在系统梳理经典车辆路径问题（VRP）向电动车辆路径问题（EVRP）的演进，重点对比充电与换电在建模假设、决策变量和约束处理上的本质差异，为后续构建带换电的EVRP数学模型及元启发式算法奠定理论基础。

## 2. 从VRP到EVRP：里程焦虑的数学引入

经典的车辆路径问题（VRP）仅考虑距离、时间窗和载重约束，假设车辆可以无限行驶。而电动车辆路径问题（EVRP）在VRP的基础上增加了**电量约束**，要求任何时刻车辆剩余电量不能低于零（或安全阈值），且必须在途中通过外部设施（充电站/换电站）补充能量。

### 2.1 基础EVRP建模要素

- **电量消耗函数**：通常假设电量消耗与行驶距离成正比，但更精确的模型会引入载重、速度、坡度等非线性因素。
- **能量补给站**：分为充电站（CS）和换电站（BSS），其位置、服务时间、可用性均为输入参数。
- **决策变量**：除了经典的弧段选择变量（\(x_{ij}\)）外，还需引入能量状态变量（\(soc_i\)）以及是否使用补给站的二元变量。

### 2.2 充电与换电的数学处理分歧

| 特征 | 充电模式 | 换电模式 |
|------|----------|----------|
| 服务时间 | 变量，依赖于充电功率、当前电量和目标电量 | 常数（例如3分钟） |
| 电量增量 | 非线性函数（通常为线性近似或分段线性） | 常数（充满至最大容量） |
| 中间状态 | 允许部分充电，电量连续变化 | 要么不换，要么换完即满 |
| 决策复杂性 | 需决定充多少电（连续优化子问题） | 仅需决定是否换电（离散决策） |
| 站内排队 | 可能长时间占用，易形成队列 | 服务时间短，队列通常可忽略 |
