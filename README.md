# Swap-N-Charge VRP: Energy-Aware Routing with Opportunistic Battery Exchange

## 📌 项目简介

本项目聚焦于电动车辆路径规划中的“里程焦虑”问题。相比传统充电，**电池换电（Battery Swapping）** 可在数分钟内恢复续航，但给路径优化带来了新的挑战：车辆何时、何地“机会性地”进行换电？你需要设计一个智能算法，在配送路径最短的同时，动态管理电池能耗，避免中途抛锚。

## 🎯 研究目标

| 优先级 | 目标 |
|--------|------|
| 主要 | 建立非线性能耗模型（考虑载重、距离） |
| 主要 | 数学建模带换电的电动车辆路径问题 (EVRP-BSS) |
| 主要 | 开发启发式/元启发式算法求解大规模实例 |
| 扩展 | 加入动态交通干扰，并实现真实地图可视化 |

## 🗺️ 任务节点（共9项，至少完成5项）

### 第一阶段：基础与能耗建模
- [ ] **Node 1**：阅读文献，撰写综述（对比充电与换电的数学差异）  
- [ ] **Node 2**：编写能耗计算模块（支持载重影响），绘制电池消耗曲线  
- [ ] **Node 3**：使用精确求解器（PuLP/Gurobi）求解小规模网络（1 depot + 2 swap stations + 8 customers）

### 第二阶段：核心算法实现
- [ ] **Node 4**：实现元启发式框架（如 ALNS、GA、模拟退火），先求解经典 VRP 基准  
- [ ] **Node 5**：融入换电逻辑（惩罚缺电路线，奖励高效换电）  
- [ ] **Node 6**：测试 50~100 客户规模的实例，输出计算时间与最优性对比报告

### 第三阶段：创新与可视化
- [ ] **Node 7**：引入随机交通延迟（动态能耗），实现算法在电池紧急情况下的重路由  
- [ ] **Node 8**：使用 Folium / OSMnx 将规划路径投影到真实城市地图（如纽约或伦敦）  
- [ ] **Node 9**：汇总所有指标（总距离、能耗、换电次数），制作学术海报
